<div class="ctcAdminInstruction">
<h4>Instructions </h4>
<div>
<ol>
<li>
Start with setting up the e-commerce site by filling up form in Business Information sub tab under Business Setting tab, which is necessary for you to continue with other steps. After doing so plugin will set up business  pages  for  you. Then fill in the form in Business sub tab which is also necessary for plugin to function properly(at least you need to fill in TAX and Currency field).  If you are going to use Strip fill in stripe Keys(if left empty option will not display in frontend).
</li>
<li>
<p>After that you need to fill in the shipping field as applicable:</p>
If you are going to use USPS fill in following field, else functionality will not work:
    <ol>
    <li>USPS API Key</li>
    <li>Shipping Weight Unit  as pound</li>
    <li>Machinable</li>
    <li>Shipping length</li>
    <li>Shipping box size</li>
   </ol>
<p><i>If left empty option  will not display in frontend</i></p>
</li>
<li>
  Then fill in email SMTP information for customer to receive  purchase confirmation.
</li>

<li>
Then add category and applicable sub categories. Then you will will be able to add products to the and  applicable other information. Doing so plugin will also create WordPresss post which is necessary if you wish to enable user to review you product.After that you will be able to see products in Products List tab where you can view and update product information as well as purge product.
</li>
<li>
Next comes Purged Product sub tab, where you can view purged products (inactive products), there you also re-add product back to you store.
</li>
<li>
Under Discount tab you can add new discount and coupon and update/delete as necessary.
</li>
<li>
  Then under Orders tab you can view pending order that you need to take care of and list of completed orders so far. 
</li>
</ol>

<p>
After this you will be able to see bunch of page generated by plugin and a CT commerce menu which will contain basic main featured product page (which displays list of featured products), Product cart page(which displays products cart) and Categories page(which display all of the product categories with active products). You can make this menu Home menu or create your own menu as you wish.Other than Current Discount , other pages are worthless showing in front end they are there to process customer request only(do not delete these pages leave them as is, when you deactivate plugin they will deleted automatically.).
</p>
</div>
</div>




<div class="ctcAdminPanelShortcodes">

<h4>Shortcodes</h4>
<div >
<p>Plugin comes with bunch of useful Shortcodes</p>
<ol>
<li>
<b>[ctcMainStorePage]</b> : you can place it anywhere in any page to display featured products
</li>
<li>
<b>[ctcDisplayProductCategorie]</b> : you can place it anywhere in any page to display product categories
</li>
<li>
<b>[ctcDisplayDiscountProducts]</b> : you can place it anywhere in any page to display all discounted product
</li>
<li>
<b>[ctcGetPostAddToCart]</b> : it adds Add To cart option to the post, if you wish to use Wordpress post created by plugin as product page(which is customisable using WordPress editor like other post) rather than default product page.
</li>
<li>
<b>[ctcGetPostRating]</b> : it adds Thumb Up/Down option to the post, if you wish to use Wordpress post created by plugin as product page(which is customisable using WordPress editor like other post) rather than default product page.
</li>
<li>
<b>[ctcPostSocialbarSharing]</b> : it adds Social sharing  option to the post, if you wish to use Wordpress post created by plugin as product page(which is customisable using WordPress editor like other post) rather than default product page.
</li>

<li>
<b>[ctcPostGalleryOverlay]</b> : it adds overlay to product gallery (created using  WordPress gallery shortcode) option to the post, if you wish to use Wordpress post created by plugin as product page(which is customisable using WordPress editor like other post) rather than default product page.
</li>
</ol>
</div>
</div>

<div class="ctcAdminPanelRestApi">
<h4>Rest API</h4>
<div>
<ul>
<li><h4><i>/ctc/v1/products</i></h4>
<ol>
<b>Response : </b>
<li><i>productId</i> : Product's Id.</li>
<li><i>productName</i> : Name of the product.</li>
<li><i>categoryName </i>: Product's Category. </li>
<li><i>primaryImage </i>: Product's primary image.</li>
<li><i>addtionalImages</i> : Product's additional images.</li>
<li><i>productPrice</i> : Product's price.</li>
<li><i>preOrder </i>: 1 if pre-order is available.</li>
<li><i>thumbsUpCount</i> : Total thumbs up for the product.</li>
<li><i>thumbsDownCount </i>: Total thumbs down for the product.</li>
<li><i>productVariation </i>:  Variation of products with invertory sperated by ':'.</li>
</ol>

</li>
<li><h4><i> /ctc/v1/posts</i></h4> 
<ol>
<b>Response : </b>
<li><i>productId</i> : Product's Id.</li>
<li><i>thumbsUpCount</i> : Total thumbs up for the product.</li>
<li><i>thumbsDownCount</i> : Total thumbs down for the product.</li>
<li><i>postPermaLink</i> : Url for the product.</li>
</ol>

 </li>
</ul>
</div>
